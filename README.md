Politique de confidentialité — multi.love
Dernière mise à jour : 25 mars 2026
1. Responsable du traitement
multi.love est opéré par Smarsys, Christophe Arn, Oulens-sous-Echallens, Suisse. Contact : contact@smarsys.com
2. Données collectées

Informations de profil (pseudo, date de naissance, genre, localisation)
Photos uploadées volontairement
Données de vérification d'identité (traitées par Stripe Identity)
Données d'utilisation et de navigation dans l'app
Identifiants d'appareils pour les notifications push

3. Utilisation des données

Fourniture et amélioration du service
Vérification de l'âge et de l'identité (majorité requise)
Envoi de notifications liées à l'activité
Traitement des paiements via RevenueCat et Stripe

4. Permissions utilisées

Caméra : prise de photos de profil et vérification d'identité
Notifications : alertes d'activité
Localisation : affichage des événements proches (optionnel)

5. Partage des données
Vos données ne sont jamais vendues. Elles peuvent être partagées avec :

Stripe (vérification d'identité et paiements)
RevenueCat (gestion des abonnements)
Infomaniak (hébergement Suisse)

6. Conservation
Les données sont conservées tant que le compte est actif. Suppression possible sur demande à contact@smarsys.com
7. Droits RGPD
Accès, rectification, suppression, portabilité — contactez contact@smarsys.com
8. Sécurité
Les photos sont stockées sur des serveurs privés Infomaniak en Suisse. Les communications sont chiffrées (HTTPS/WSS).
9. Âge minimum
multi.love est réservé aux personnes majeures (+18 ans). La vérification d'identité est obligatoire pour accéder aux fonctionnalités complètes.
